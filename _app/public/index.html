Washington Post data on fatal police shootings

This project uses the data provided by the Washington Post for all of its charts.

The data represents fatal police shootings, and looks at a number demographic factors.

I am not the creator of this data set, I am simply visualizing it.

This project is a work in progress. Not all of the data is represented yet, but you can see the roadmap, peep the code, and/or help out with some charts on Github.

If the project isn't working for you, try using Google Chrome. If you're already using it, upgrade it—this page uses modern JavaScript and requires a modern browser.
